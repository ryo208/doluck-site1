# Do Luck 道楽 — 公式サイト

北海道・十勝 池田町の喫茶と旅人宿 [Do Luck 道楽](https://ryo208.github.io/doluck-site1/) のウェブサイトです。

このREADMEは、**オーナー自身がちょっとした更新をするためのガイド**です。

---

## しくみ（30秒で理解）

- サイトは素のHTMLだけでできています。サーバーもデータベースもありません。
- `master` ブランチにファイルを保存（コミット）すると、**1〜2分で自動的に公開**されます（GitHub Pages）。
- 反映されたかは、リポジトリの **Actions** タブの「pages build and deployment」が緑✓になったかで確認できます。公開後もブラウザのキャッシュで数分古いまま見えることがあるので、スーパーリロード（Shift+再読み込み）してください。

## 編集のやり方は2つ

### A. いちばん手軽：ブラウザだけで編集（インストール不要）

1. GitHubでこのリポジトリを開き、キーボードの **「.」（ピリオド）** を押す
2. ブラウザ上でVS Codeが開くので、ファイルを直して保存
3. 左端の「ソース管理」アイコン → 変更内容にひとことメッセージを書いて「コミットしてプッシュ」
4. 1〜2分待てば公開

### B. しっかりやる：VS Codeを自分のPCに入れて編集

1. [VS Code](https://code.visualstudio.com/) と [Git](https://git-scm.com/) をインストール
2. ターミナルで `git clone https://github.com/ryo208/doluck-site1.git`
3. VS Codeでフォルダを開くと、拡張機能 **Live Server** のインストールを勧められるので入れる
4. `index.html` を右クリック →「Open with Live Server」で、保存するたびブラウザに即反映されるプレビューが見られる
5. 良ければ「ソース管理」からコミット＆プッシュ（＝公開）

## よくある更新と、その場所

| 更新したいこと | 場所 | 難易度 |
|---|---|---|
| Googleアナリティクスの測定ID | `js/analytics.js` の `G-XXXXXXXXXX` を書き換えるだけ（1ファイル） | ★ かんたん |
| 写真の差し替え | `images/` に**同じファイル名**で上書きする（例: `photo-room1.jpg`） | ★ かんたん |
| 営業時間・定休日・料金・電話番号 | 下の「⚠ 大事な注意」参照。全ページに複製されているので**全ファイル一括置換**で直す | ★★ 置換を使えばOK |
| 文章の言い回し・デザイン・新ページ | 10言語×6ページに影響するため手作業は事故のもと | ★★★ Claudeに頼む |

### ⚠ 大事な注意：同じ内容が「10言語ぶん」複製されています

このサイトは日本語のほかに `en/` `ko/` `zh-hans/` `zh-hant/` `de/` `it/` `ru/` `fr/` `es/` の9言語のページを持っていて、**同じ情報がそれぞれのファイルに入っています**（例：営業時間「10:00 – 18:00」は約20ファイルに登場します）。

日本語ページだけ直すと、外国語ページが古い情報のまま残ります。数字や時間など**言語に関係ない値**を直すときは、VS Codeの**全ファイル検索・置換**を使ってください：

1. `Ctrl+Shift+H`（Macは `Cmd+Shift+H`）で「検索と置換」を開く
2. 検索欄に今の値（例: `10:00 – 18:00`）、置換欄に新しい値
3. ヒットした件数と場所を確認してから「すべて置換」

**文章そのもの**（キャッチコピーや説明文）は言語ごとに翻訳されているので、一括置換では直せません。それはClaudeに「◯◯を△△に変えて」と頼むのが安全で確実です。

## 困ったら

- 保存したのに反映されない → Actionsタブが緑✓か確認 → スーパーリロード
- 壊してしまったかも → 慌てなくて大丈夫。Gitはすべての履歴を保存しているので、Claudeに「さっきの変更を元に戻して」と言えば戻せます
- 複数ファイルにまたがる変更・デザイン変更 → Claudeに依頼するのが早くて安全です
